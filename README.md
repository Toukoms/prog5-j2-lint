# PROG5 LINT

Dans ce repository j'ai utilisé checklint pour verifier la convention de nom sur java

il y a 3 cas qui peut être apperçu dans les logs de github actions:

1. Tous les **class** doivent être PascalCase
2. Tous les **variables** doivent être camelCase
3. Tous les **methodes** doivent être camelCase
